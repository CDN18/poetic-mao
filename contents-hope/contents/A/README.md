# 古体诗词

> 关于诗，有三条：（一）精炼；（二）有韵；（三）一定的整齐，但不是绝对的整齐……诗必须有诗意，要含蓄。
>
> ——毛泽东《诗人要向民间学习》（1957年1月14日下午同诗人臧克家、袁水拍谈诗歌创作问题）
